## 项目介绍

基于 Next.js 构建的实时聊天应用，支持多用户实时通信。采用简洁的后端设计和清晰的代码结构。

## 主要特性

### 技术栈

- NextJS + TailwindCSS + TypeScript
- 使用 bun 进行包管理
- Shadcn/ui 组件库

### 后端集成

- 实时数据库
- Convex 后端 API
- 完整的 CRUD 操作

### 用户认证

- 密码登录
- GitHub OAuth
- Google OAuth

### 聊天功能

- 私聊和群聊
- 图片发送
- 表情回应
- 消息回复
